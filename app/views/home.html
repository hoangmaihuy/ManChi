What is ManChi?

It's a shorthand of Mandarin Chinese, a simple web application built on AngularJS, Bootstrap and gh-pages for hosting.

What can ManChi do?

First of all, this is an alpha version and I have many ideas, features and tweaks to develope in the near future. Forgive me if this application retards.

This application concept is from the dictation test on my Chinese class that students have to hear the pronunciation and write chinese characters (Hanzi) into paper.

And that's why ManChi's Writing is here to help, reading out loud the pronunciation and checking if you write it correctly.

How can I learn Chinese characters?

All the new words and writing words are from a book called "Hanyu Jiaocheng".
If you are learning that book so you can practice writing character by this application.

How can I type Chinese characters?

It's a non-sense question if you are Chinese, just skip it.

You can use Pinyin input keyboard but I don't recommend because this application would be helpless using that method. My concept creating this project is using handwriting input methods. Just draw the right strokes and they will do the rest for you.

You can follow the instruction here to install handwriting keyboard on your platform.
Or if you're using an Android phone, just download Google Handwriting Input.
